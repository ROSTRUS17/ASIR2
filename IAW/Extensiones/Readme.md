## Python
![Imagen Python](/IAW/Extensiones/imagenes/Python.png)
**Descripción:**  
La extensión oficial de **Python** para Visual Studio Code es imprescindible.  
Incluye soporte para **ejecución de scripts, depuración, linting, IntelliSense, formateo de código, testeo con pytest/unittest** y mucho más.  
Es el núcleo sobre el que trabajan las demás extensiones de Python.

Elijo esta extensión por la facilidad de probar y ejecutar scripts directamente en VScode.

---

## Pylance
![Imagen Pylance](/IAW/Extensiones/imagenes/Pylance.png)

**Descripción:**  
**Pylance** mejora enormemente la experiencia de desarrollo en Python al proporcionar un **motor de análisis estático muy rápido y preciso**.  
Permite **autocompletado avanzado, navegación por definiciones, detección de errores en tiempo real y mejor soporte para tipado estático (type hints)**.  
Hace que Visual Studio Code se acerque a la potencia de un IDE completo. 

Elijo esta extensión por la facilidad de autocompletado avanzado que provee.

---

## Jupyter
![Imagen Jupyter](/IAW/Extensiones/imagenes/Jupyter.png)

**Descripción:**  
La extensión de **Jupyter** integra cuadernos interactivos directamente en Visual Studio Code.  
Con ella puedes **crear, abrir y ejecutar notebooks (.ipynb)** sin salir del editor.  
Es ideal para **ciencia de datos, machine learning y análisis exploratorio** ya que combina código, visualizaciones y texto explicativo en un mismo entorno.
Elijo esta extensión por el analisís de datos y documentación interactiva, que será útil para el big data más alante en este curso.
