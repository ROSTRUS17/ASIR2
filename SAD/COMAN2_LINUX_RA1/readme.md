1️⃣ ¿Qué sucede si un usuario fuera del grupo devEria2 intenta acceder al directorio?

➡️ No podrá entrar ni listar su contenido.
Recibirá un error como:

* bash: cd: /home/devEria2/di_recto: Permiso denegado

2️⃣ ¿Qué sucede si tronko2 intenta modificar un archivo dentro del directorio?

➡️ No podrá hacerlo, porque el grupo solo tiene lectura y ejecución (r-x).
Intentar editar o crear un archivo dará:

* bash: permiso denegado
