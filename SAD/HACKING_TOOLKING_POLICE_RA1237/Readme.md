# ¿Qué es AbuseIPDB?

 AbuseIPDB es un proyecto dedicado a ayudar a combatir la propagación de piratas informáticos, spammers y actividades abusivas en Internet. 

 Nuestra misión es ayudar a que la Web sea más segura proporcionando una lista negra central para que los webmasters, administradores de sistemas y otras partes interesadas informen y encuentren direcciones IP que se hayan asociado con actividades maliciosas en línea. 

 Sirve para informar una dirección IP asociada con una actividad maliciosa o verificar si se ha informado una dirección IP utilizando el cuadro de búsqueda de arriba.

# URLVoid

Múltiples listas de bloqueo Con URLVoid puedes analizar un sitio web a través de motores de listas de bloqueo 30+ y servicios de reputación de sitios web en línea para facilitar la detección de sitios web fraudulentos y maliciosos.

Identificar sitios web involucrados en incidentes de malware y phishing. Análisis de amenazas Las empresas de seguridad cibernética y los investigadores de TI utilizan URLVoid para acelerar el proceso de análisis de amenazas cibernéticas; puede identificar mejor sitios web potencialmente maliciosos que han sido clasificados como una amenaza por múltiples fuentes confiables. 

Informe de seguridad: Cuando envía un sitio web, puede ver su informe de seguridad, que incluye nombres de listas de bloqueo utilizadas para escanear y un enlace a su informe respectivo, detalles sobre la dirección IP del sitio web, la fecha de creación del dominio, la ubicación del servidor y más.

# Metadefender

MetaDefender Cloud es una herramienta gratuita de escaneo de archivos y análisis de malware en línea.

Multiescaneo : Utiliza más de 20 motores antivirus (como McAfee y Bitdefender) en un solo escaneo para una detección avanzada de amenazas.

Análisis de archivos, URL, IP o hash: Permite subir o buscar cualquiera de estos elementos para su análisis.

Content Disarm and Reconstruction (CDR): Elimina amenazas ocultas de documentos manteniéndolos utilizables, a diferencia de los escáneres típicos.

Escaneo de vulnerabilidades: Detecta riesgos conocidos en archivos, IPs, URLs y hashes de archivos.
